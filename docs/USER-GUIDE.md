# MeowCRM 使用手冊

## 快速開始
1. 前往 https://crm.meowcloud.ai
2. 點擊「免費開始」註冊帳戶
3. 登入後進入 Dashboard

## 功能說明

### 客戶管理
- 新增客戶：點擊客戶頁面「新增客戶」按鈕
- 編輯客戶：點擊客戶名稱進入詳情頁
- 搜尋/篩選：使用列表上方的搜尋框

### 聯絡人
- 每位客戶可關聯多位聯絡人
- 在客戶詳情頁的「聯絡人」tab 管理

### 互動時間軸
- 記錄每次客戶互動（會議、電話、Email）
- 在客戶詳情頁的「活動」tab 查看

### 任務看板
- Kanban 式任務管理（待辦/進行中/完成）
- 可關聯特定客戶
- 到期提醒在 Dashboard 顯示

### Dark Mode
- 點擊 Header 右上角的 🌙/☀️ 圖示切換
