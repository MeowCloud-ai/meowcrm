# PRD — MeowCRM 產品需求規格

> **版本：** v0.1 | **日期：** 2026-02-19 | **撰寫：** 阿福（AI CEO）

## 產品願景

你專心開會，我幫你管客戶。

## 產品定位

MeowCRM 是一個為「懶得用 CRM 的人」設計的自動導航客戶管理系統。核心價值不在 CRM 功能的豐富度，而在於**零手動輸入**——會議結束後，客戶資訊自動整理好。

**戰略定位：** MeowCRM 不是利潤中心，是數據飛輪的入口。重點在累積用戶忠誠度與結構化商業數據，為未來 MeowInsights / MeowEngine 提供燃料。

## 目標用戶（ICP）

- 5-20 人專案制公司的老闆 / 業務負責人
- 透過 BNI / 公會 / 商會活動頻繁開會的中小企業主
- 痛點：Salesforce 太貴、HubSpot 太複雜、Excel 管客戶管到崩潰
- 特徵：**不想學 CRM，只想客戶資料自動整理好**

## 核心功能

### F1: MeowMeet 智慧歸檔（核心中的核心）

會議結束後，MeowMeet 的 AI 摘要自動進行實體辨識（NER），擷取：
- 公司名稱
- 聯絡人姓名 / 職稱
- 行動項目（Action Items）+ 負責人 + 截止日
- 關鍵決策
- 潛在商機金額（若有提及）

**UX 流程：**
1. MeowMeet 摘要完成後，彈出「智慧歸檔卡片」
2. 卡片顯示辨識結果，用戶可快速確認/修改
3. **Human-in-the-loop**：用戶的每次修正都記錄為 NER 訓練數據，持續優化辨識品質
4. 一鍵「同步到 MeowCRM」（透過 Supabase Edge Function，不直接存取 DB）
5. 系統自動：
   - 建立或更新客戶 / 聯絡人（含模糊比對去重建議）
   - 附加會議記錄為互動日誌
   - 將行動項目轉為 CRM 任務

### F2: 客戶管理（極簡）

- 客戶列表（公司名、產業、狀態）
- 客戶詳情頁：基本資訊 + 互動時間軸 + 相關商機
- 搜尋 / 篩選
- **不做**：自訂欄位、複雜標籤系統

### F3: 聯絡人管理

- 每個客戶底下的聯絡人列表
- 聯絡人：姓名、職稱、Email、電話
- 從 MeowMeet 自動辨識建立
- **不做**：社群帳號整合、名片掃描

### F4: 商機管理（極簡看板）

- 看板式商機管理（Kanban）
- 預設階段：接觸 → 提案 → 報價 → 成交 / 失敗
- 每個商機：金額、預計成交日、關聯客戶、備註
- 從會議中自動建議建立商機
- **不做**：複雜銷售流程、多管線、預測模型

### F5: 任務管理

- 從會議行動項目自動生成
- 任務：標題、負責人、截止日、關聯客戶
- 簡單狀態：待辦 / 進行中 / 完成
- 到期提醒（Email）
- **不做**：子任務、依賴關係、甘特圖

### F6: 互動日誌

- 每次 MeowMeet 會議自動成為一條互動記錄
- 可手動新增互動（電話、Email、拜訪）
- 每條記錄可連回 MeowMeet 完整逐字稿
- 客戶頁面顯示完整互動時間軸

## 非功能需求

### 技術架構
- **前端：** Next.js（Web App）
- **後端：** Supabase（PostgreSQL + Auth + Realtime）
- **部署：** Vercel
- **整合：** MeowMeet API（智慧歸檔數據同步）

### 效能
- 頁面載入 < 2 秒
- 智慧歸檔同步 < 5 秒
- 支援 10,000+ 客戶資料

### 安全
- Supabase Auth（Email + Google OAuth）
- Row Level Security（RLS）確保租戶隔離
- 商業數據加密存儲
- 台灣個資法合規（隱私政策 + 數據使用同意）

### 使用者體驗
- 註冊到建立第一個客戶 < 3 分鐘
- 零學習曲線（看了就會用）
- Mobile responsive（但不做獨立 App）

## 關鍵指標（KPI）

> 注意：MeowCRM 的 KPI 不是營收，是數據量。

| 指標 | 目標（6 個月） |
|------|----------------|
| 註冊企業數 | 500+ |
| 活躍企業數（月） | 200+ |
| CRM 客戶資料筆數 | 10,000+ |
| 智慧歸檔使用率 | > 60% 的會議 |
| NER 準確率 | > 85% |

## 數據模型（初步）

```
Organization (租戶)
  ├── User (使用者)
  ├── Customer (客戶/公司)
  │     ├── Contact (聯絡人)
  │     ├── Deal (商機)
  │     ├── Task (任務)
  │     └── Activity (互動日誌)
  │           └── → MeowMeet Meeting (外部連結)
  └── Settings
```

## 與 HubSpot Free CRM 的差異

| 面向 | HubSpot Free | MeowCRM |
|------|-------------|---------|
| 數據輸入 | 手動輸入為主 | 會議自動擷取為主 |
| 目標用戶 | 會用 CRM 的人 | 懶得用 CRM 的人 |
| 複雜度 | 功能多但上手難 | 極簡，看了就會 |
| 免費版目的 | Upsell 漏斗 | 數據收集策略 |
| 中文支援 | 有但非優先 | 繁中原生 |
| 會議整合 | 無深度整合 | MeowMeet 一鍵歸檔 |
